## I.语法规范
主要由两个部分构成：选择器以及一条或多条声明
写在< head >< /head >中，以< style >标签表样式修改
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        p {color:red; }
    </style>
</head>
<body>
    <p>红色</p>
</body>
</html>
```
